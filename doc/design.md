# 1 项目概述

## 1.1 项目背景

在高校及教育机构中，教师评选、职称评审、教学成果申报等工作需要提交大量申报材料。传统方式主要通过：

- Word文档填写
- 邮件或纸质提交
- 人工整理材料
- 手动审核流程

这种方式存在以下问题：

1. 材料格式不统一
2. 信息重复填写
3. 审核流程效率低
4. 材料整合困难
5. 数据难以沉淀和复用

为解决上述问题，开发**教师评选申报材料管理系统**，实现申报流程的信息化、规范化和自动化。

---

## 1.2 建设目标

系统建设目标包括：

1. 建立统一申报平台
2. 实现在线材料填写
3. 实现材料模板统一管理
4. 自动整合生成申报材料
5. 建立多级审批流程
6. 实现材料电子归档

---

## 1.3 系统范围

系统主要支持以下业务场景：

- 教师评优评先申报
- 职称评审材料申报
- 教学成果奖申报
- 科研成果申报

---

## 1.4 用户角色

| 角色       | 说明                   |
| ---------- | ---------------------- |
| 教师       | 填写申报材料、上传附件 |
| 部门管理员 | 审核部门教师申报       |
| 学校管理员 | 管理系统、项目、模板   |
| 专家评审   | 查看材料并评审         |

# 2 系统总体设计

## 2.1 系统架构

系统采用 **B/S（Browser/Server）架构**。

系统架构如下：

```
浏览器
   │
   │ HTTP/HTTPS
   │
前端应用层
(React + Vite + Ant Design)
   │
   │ REST API
   │
后端服务层
(Python FastAPI)
   │
 ┌───────────┬───────────┐
数据库       文件存储
(MySQL)      (本地文件系统)
```

---

## 2.2 技术架构

| 层级     | 技术                                  |
| -------- | ------------------------------------- |
| 前端     | React 18 + Vite + Ant Design 5 + TypeScript |
| 后端     | Python 3.11 + FastAPI                 |
| 数据库   | MySQL 8.0                             |
| ORM      | SQLAlchemy 2.0                        |
| 数据迁移 | Alembic                               |
| 认证     | JWT (python-jose)                     |
| 文件存储 | 本地文件系统（可扩展至OSS）            |
| 部署     | Docker                                |

# 6 接口设计

基于 FastAPI 框架，所有接口前缀为 `/api`，自动生成 Swagger 文档（`/docs`）。

---

## 6.1 认证接口

| 方法 | 路径 | 说明 |
| ---- | ---- | ---- |
| POST | /api/auth/login | 用户登录 |
| POST | /api/auth/register | 用户注册 |

### 登录请求

```json
POST /api/auth/login
{
  "username": "admin",
  "password": "123456"
}
```

### 登录返回

```json
{
  "access_token": "eyJhbGciOiJIUzI1NiIs...",
  "token_type": "bearer",
  "user": {
    "id": 1,
    "username": "admin",
    "name": "管理员",
    "role": "school_admin"
  }
}
```

---

## 6.2 核心接口列表

| 模块 | 方法 | 路径 | 说明 |
| ---- | ---- | ---- | ---- |
| 用户 | GET | /api/users/me | 获取当前用户 |
| 用户 | GET | /api/users | 用户列表（管理员） |
| 项目 | GET | /api/projects | 项目列表 |
